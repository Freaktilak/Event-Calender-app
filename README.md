# Event-Calender-app
In this project we are going to develop a Event Calendar App using HTML, CSS and JAVA Script. Calendar apps are also frequently called scheduling,
productivity, and time-management apps. 
We can plan a whole year using this Calendar App. The main objective of this project is
“There Are Dreams And There Are Planners; The Planners Make Their Dreams Come True”. So we are going to create a project to achieve your Dreams.
